# Gemini Simplify Extension 🚀

This extension for the Gemini CLI adds the `/simplify` command, an elite orchestrator that refactors code based on real-time documentation and specialized agent analysis.

## Installation

```bash
gemini extensions install https://github.com/your-username/gemini-simplify
```

## What does it do?

- **Dynamic Research:** Detects your ecosystem (Python 3.12, Rust, Node, etc.) and searches for current best practices.
- **Parallelism:** Invokes 4 specialized agents in parallel:
  - `@abstraction`: Identifies patterns and code duplication.
  - `@clarity`: Focuses on naming and reducing nesting.
  - `@performance`: Optimizes loops and allocations.
  - `@standards`: Ensures modern conventions (PEP8, Typing, etc.).
